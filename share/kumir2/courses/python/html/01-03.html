Имена

Для хранения вычисленных значений можно использовать именованные переменные. Эти имена можно затем использовать в других выражениях.

Отменить задание имени можно с помощью оператора del

x = 2 + 2
# сейчас в x хранится
# значение 4

y = x / 2
# в y хранится вещественное 
# значение 2.0

print(y) 
# вывод: значение (2+2)/3

y = 2
# а теперь в y 
# хранится целое значение 2

del x, y
# теперь имен x и y
# не существует!

print(x)
# ОШИБКА ВЫПОЛНЕНИЯ

Присваивания значений нескольких имен можно выполнять в одну строку, через запятую:

a, b = 3, 14159
# a = 3
# b = 14159
