Списки

Объявление списка

Список - это упорядоченная последовательность любых набор данных.

Список объявляется в квадратных скобках; разделителем между элементами списка является запятая.

a = [1, "строка", 3.14159, 567, "еще строка"]

Конкатенация списков

Конкатенация, или другими словами, – слияние списков, выполняется с помощью оператора +.

a = [1, 2, 3]
b = [4, 5, 6]
print(a + b)
# вывод: [1, 2, 3, 4, 5, 6]

Операция конкатенации применима только для двух списков. Чтобы добавить в список один элемент, нужно "прибавить" к списку новый список, состоящий из одного элемента:

a = [1, 2, 3]

a = a + 4
# ОШИБКА: так нельзя

a = a + [4]
# a = [1, 2, 3, 4]

z = 0
a = [z] + a
# a = [0, 1, 2, 3, 4]

Обращение к отдельному элементу списка

Для получения отдельного элемента списка используются квадратные скобки [].

Нумерация элементов начинается с нуля!

Можно использовать отрицательные значения индекса – в этом случае будет получен элемент с конца списка.

squares = [1, 4, 9, 16, 25]

first = squares[0]
# first = 1

last = squares[-1]
# last = 25

В отличии от строк, списки являются изменяемыми объектами. Можно присваивать значения отдельным элементам существующего списа.

При этом нужно следить за правильностью индексов: нельзя изменить элемент, которого в списке не существует!

Следует помнить, что присваиванием списка какой-либо переменной не копирует его содержимое, а присваивает только ссылку на существующий список.

a = [1, 4, 9, 16, 25]

a[1] = 2
# a = [1, 2, 9, 16, 25]

b = a
# b = [1, 2, 9, 16, 25]

b[0] = 100
print(a)
# вывод: [100, 2, 9, 16, 25]
# потому что a и b - это
# один и тот же список

a[10] = 10
# ОШИБКА ВЫПОЛНЕНИЯ:
# в списке нет элемента 
# с индексом 10

Выделение срезов

Срезом называется фрагмент списка.

Для получения срезов испольуются квадратные скобки с указанием двух границ, между которыми стоит символ двоеточия :, точно так же, как для строк.

squares = [1, 4, 9, 16, 25]

sub = squares[1:-1]
# sub = [4, 9, 16]

Полученный срез является новым списком:

squares = [1, 4, 9, 16, 25]
sub = squares[1:-1]

sub[1] = 100
print(squares)
# вывод: [1, 4, 9, 16, 25]
# потому что sub и squares - это
# уже разные списки

Изменение фрагмента списка

Если в левой части присваивания указан срез, а в правой части – другой список, то в исходном списке заменяются несколько элементов. При этом, размер изменяемой части и нового списка может не совпадать:

s = [1, 4, 9, 16, 25]
s[1:3] = ['a', 'b', 'c', 'd']
print(s)
# вывод: [1, 'a', 'b', 'c', 'd', 16, 25]

В приведенном выше примере происходит следующее:

  1. Из списка s удаляются элементы от индекса 1 (включительно) до индекса 3 (не включая элемент с индексом 3)
  2. Вместо удаленных элементов вставляются элементы из списка в правой части присваивания (которых в данном примере больше, чем удаленных).

Длина списка

Встроенная функция len(список) используется для получения количества символов в списке:

l = [1, 4, 9, 16, 25]
print( len(l) )
# вывод: 5
