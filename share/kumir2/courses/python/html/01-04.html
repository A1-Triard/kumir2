Ввод строки и операции над строками

Ввод строки из терминала

Для ввода строки из терминала используетя оператор input().

line = input()
# ввод одной строки
# (до нажатия ENTER) в переменную 
# с именем line

name = input("Как тебя зовут:")
# вывод приветствия "Как тебя зовут: ",
# после этого - чтение в переменную
# name строки от двоеточия (последний
# символ приветствия) до нажатия ENTER

Конкатенация

Конкатенация, или другими словами, – слияние строк, выполняется с помощью оператора +.

a = "Привет"
b = "Мир"
print(a + ", " + b + "!")
# вывод: Привет, Мир!

Повторение

Оператор * используется для повторения строки несколько раз:

s = "мяу!"
n = 5

ss = n * s
# ss = "мяу!мяу!мяу!мяу!мяу!"

tt = s * (n - 2)
# tt = "мяу!мяу!мяу!"

Получение отдельного символа строки

Для получения отдельного символа строки используются квадратные скобки [].

Нумерация символов начинается с нуля!

Можно использовать отрицательные значения индекса – в этом случае будет получен символ с конца строки:

s = "123456789абвг"
print(s[0])
# вывод: 1
# '1' - "нулевой" символ в строке

print(s[1]
# вывод: 2

print(s[-1])
# вывод: г
# 'г' - последний символ в строке

print(s[-2])
# вывод: в
# 'в' - предпоследний символ в строке

Получение фрагмента строки

Для получения подстроки испольуются квадратные скобки с указанием двух границ, между которыми стоит символ двоеточия :.

Нумерация символов начинается с нуля; можно использовать отрицательные индексы для определения границы с конца строки.

Правая граница указывается не включая последний символ! Разница между правой и левой границей – это длина полученной подстроки.

Если не указана первая граница, то подразумевается начало исходной строки, если не указана вторая граница – конец строки.

word = "программирование"

a = word[0:2]
# a = "пр"

b = word[5:6]
# b = "а"

c = word[:5]
# c = "прогр"
# это эквивалентно word[0:5]

d = word[5:]
# d = "аммирование"
# это эквивалентно word[5:16]

e = word[:]
# e = "программирование"
# это эквивалентно word[0:16]

f = word[1:-1]
# f = "рограммировани"
# строка содержит все символы,
# кроме первого и последнего
# из исходной строки

Неизменяемость строк

Строки в Python являются незменяемыми объектами. Для того, чтобы изменить часть строки, нужно построить новую строку из фрагментов исходной.

s = "превед"
# нужно получить s = "привет"

s[2] = "и"
# ОШИБКА ВЫПОЛНЕНИЯ:
# s - неизменяемый объект

s[3:] = "ивет"
# ОШИБКА ВЫПОЛНЕНИЯ:
# s - неизменяемый объект

s = s[:2] + "и" + s[3:-1] + "т"
# s = "привет"

Длина строки

Встроенная функция len(строка) используется для получения количества символов в строке:

s = "программирование"
print( len(s) )
# вывод: 16
