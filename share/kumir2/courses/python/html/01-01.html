Простейшие инструкции

Комментарии

Все, что начинается с символа # (решётка, в английской раскладке клавиатуры Shift+3), является комментарием до конца строки.

В комментариях можно писать все что угодно – их содержимое игнорируется.

Вывод значений на экран

Для вывода значений на экран используется встроенная функция print()

Пример: 

print(10)
# выведет на экран строку "10"

print("Привет, Мир!")  
# выведет на экран строку "Привет, Мир!"

В скобках могут быть указаны не только значения, но и выражения:

print(1+1)  
# выведет на экран сумму 1+1 -- строку "2"

Арифметические операции

Сумма, разность, произведение, частное и возведение в степень

Следующие символы используются в качестве символов операций:
СимволОперация
x + yСумма x и y
x - yРазность между x и y
x * yПроизведение x и y
x / yЧастное от деления x на y
x ** yЧисло x в степени y

Вещественное и целочисленное деление

Результат деления с помощью оператора / – это вещественное десятичное число

Очень часто в задачах требуется целочисленное деление, то есть деление с отбрасыванием дробной части. Это осуществляется с помощью оператора // (две косые черты).

Скобки внутри выражений

Для задания порядка выполнения арифметических выражений можно использовать круглые скобки.

Пример: 

print( (1 + 2) * 3 )
Здесь сначала вычисляется сумма чисел 1 и 2, затем полученный результат умножается на 3, а после этого – передается в функцию print(), которая выводит результат на экран.
