Строки

Строковые константы

Строки задаются в одинарных или двойных кавычках.

print('Это строка')
# вывод: Это строка

print("Это'тоже'строка")
# вывод: Это'тоже'строка

print('Строка с " кавычкой')
# вывод: Строка с " кавычкой

При создании строк символ \ имеет специальное значение:

КодЗначение
\\Вывод символа \
\nПереход на следующую строку
\Признак того, что продолжение текста в коде программы располагается на следующей строке

Пример:

print('Строка 1\nСтрока 2')
# вывод:
# Строка 1
# Строка 2

print('Длин\
ное слово')
# вывод: Длинное слово (в одну строку)

print('C:\\Windows\\System32')
# вывод: C:\Windows\System32

Можно задавать фрагменты текста из нескольких строк с помощью "тройных" кавычек: ''' или """

print('''\
Это очень большой текст,
  который содержит переносы строк.
   Внутри можно использовать 
      любые кавычки, 
       " ' """
кроме тех, что использованы 
для границ строки''')
